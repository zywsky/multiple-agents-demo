# Card Component

AEM Card component with image, title, description, tags, and optional button.

## Files

- `card.html` - HTL template
- `_cq_dialog/.content.xml` - Dialog configuration
- `card.js` - JavaScript for interactions
- `card.css` - Component styles
- `CardModel.java` - Sling Model
- `.content.xml` - Component definition
- `i18n/en.properties` - i18n dictionary

## Features

- Image with lazy loading
- Title and description
- Tags (multifield)
- Optional badge
- Link with configurable target
- Embedded button component (dependency)
- i18n support
- Responsive design

## Dependencies

- `example/components/button` - Button component
